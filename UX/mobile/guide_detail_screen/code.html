Custom instructions allow you to tailor ChatGPT to your specific needs, ensuring that every response you get is formatted exactly how you like it. Whether you're a coder, a writer, or a project manager, setting these up once can save you hours of repetitive typing.

# Setting up parameters

By defining a persona or a set of rules, you reduce the need for context-setting in every new chat. It acts as a system-level prompt that persists across all your interactions.

Pro Tip

Be specific about the tone. "Professional but friendly" yields better results than just "Professional".

# Advanced Examples

Here is a simple Python snippet you can use to test if your instructions for code formatting are working correctly.

test_script.py 
def greet_user(name):
    """
    Simple greeting function
    """
    return f"Hello, {name}! Welcome."

print(greet_user("Alex"))

Notice how the docstring is preserved? That's a result of the specific instruction set we configured earlier in the settings panel.
